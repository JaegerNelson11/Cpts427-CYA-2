# Career Development Experience: Job Applications

## Adventure Description
For this Choose Your Own Adventure assignment, I went with a targeted application sprint for full time technical roles and high impact internships to prepare for my upcoming Spring 2026 graduation. I completed 12 total applications across diverse sectors including consulting, government, agriculture, and manufacturing. The end-to-end process was documented to build a comprehensive professional portfolio.

## Career Goals Context
My short term goal is to secure a software engineering or database management position where I can immediately apply my experience in full-stack development, database architecture, and object-oriented programming. My long term goal is to lead technical projects and step into senior architectural roles within the tech industry, specifically focusing on creating scalable systems, reliable software, and clean code deployment.

## Deliverable Summary
This repository contains the documentation and proof of my application process. 
* **Evidence Folder:** Contains screenshots of the job description, the completed application form, and the submission confirmation for the 12 roles.
* **Jaeger_Nelson_Resume.pdf:** My personal resume used and tailored for these applications.
* **Reflection.md:** A detailed report analyzing this career development exercise.
